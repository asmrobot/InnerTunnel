需要部署两端，一个放在公网服务器上叫A端(InnerTunnel.Agent)，别一个放在需要提供真实服务的机器上叫c端(InnerTunnel.Client))。

原理
1.S端启动后连接A端。
2.A端开启两个端口，一个受理C端的连接，一个受理用户端的连接。当用户端向A端发送数据时转发到C端上。
3.C端收到A端数据后转发至真实的服务端口。

实例
w要在公司访问家里电脑的远程桌面，并且假设w有一台外网独立ip服务器，但w家用的联通宽带，联通给w家分配的是内网IP，使用本解决方案的流程为：

1.在服务器上布署A端，配置文件中FromPort=13389,ToPort=23389。
2.家里的机器安装C端，配置文件中AgentPort=23389,ServicePort=3389
3.w在公司用mstsc命令连接A端的13389
